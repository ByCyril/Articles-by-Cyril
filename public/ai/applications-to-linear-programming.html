Applications to Linear Programming in Artificial Intelligence

Introduction

According to Wikipedia, artificial intelligence is a subject in computer science where intelligence is demonstrated by computers. In my opinion, I believe artificial intelligence is more of a topic for mathematics than computer science for its heavy reliance in mathematics. Take an artificial neural network as an example. An artificial neural network is a network of interconnected nodes that replicate the behavior of the human brain. Since computers don't have a brain that is as complex as the human brain, we barrow topics from mathematics to help us represent this type of behavior in computers.

Linear programming is the heart of an artificial neural network. A linear program consists of a real valued linear function of n independent real variables with a system of finitely many linear equations and linear inequalities. In an artificial neural network, we can barrow that same theory by representing it in a form of a matrix equation.

A linear matrix equation is followed by the model Ax = b where our constant matrix or our input is denoted as A, our weights is denoted as x and our output is denoted as b

There are four steps to a neural network
• Get the dot product of our inputs and random weights
• Feed the sum through an activation function
• Calculate the error from the known outputs
• Update the weights from the calculated error

We repeat this same process n number of times. Too high, we run the risk of over-fitting, too low we would not get the optimal weights.

The activation function is selected based on the desired output. If we want an output that is between 0 and 1, we use the sigmoid function also known as the softmax function. An output between -1 to 1, we use the tanh function . An output of 0 to infinity, we use the ReLU function

There are many forms of neural networks. The one described above is known as a feed forward neural network because data is passed through in one direction. This model can easily be used for any linear programming problem from linear regression to logistic regression.
